How It Works

This extension uses a research-based estimator trained on graded samples. It analyzes tweets across 4 key dimensions:

  • Vocabulary Sophistication (35%): Estimates word difficulty from length and syllable complexity using Age of Acquisition proxies
  • Lexical Diversity (25%): Measures word variety using Type-Token Ratio (TTR)
  • Sentence Complexity (20%): Analyzes average words per sentence and sentence structure
  • Grammatical Precision (20%): Evaluates syntax complexity via punctuation patterns and clause usage
