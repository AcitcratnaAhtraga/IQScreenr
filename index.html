IqScreenr

Analyze, Filter and Guess verbal IQ of tweets

Analyze Tweet Intelligence in Real-Time

IqScreenr analyzes tweet text using a research-based estimator trained on graded samples. It evaluates tweets across four key dimensions: Vocabulary Sophistication, Lexical Diversity, Sentence Complexity, and Grammatical Precision.

Works on Chrome, Edge, Brave, and other Chromium browsers

Installation Steps:

  1. Click the "Download Extension" button below to download the extension
  2. Extract the ZIP file to a folder on your computer
  3. Open Chrome and go to chrome://extensions/
  4. Enable "Developer mode" (toggle in top-right corner)
  5. Click "Load unpacked" and select the extracted folder
  6. Done! Visit Twitter/X to see IQ badges on tweets

How It Works

IqScreenr uses a research-based estimator trained on graded samples. It analyzes tweets across four key dimensions:

  • Vocabulary Sophistication (35% weight): Estimates word difficulty from length and syllable complexity using Age of Acquisition proxies.
  • Lexical Diversity (25% weight): Measures word variety using Type-Token Ratio (TTR).
  • Sentence Complexity (20% weight): Analyzes average words per sentence and sentence structure.
  • Grammatical Precision (20% weight): Evaluates syntax complexity via punctuation patterns and clause usage.

üîí Fully client-side: All processing happens in your browser. No data is sent to any server.
